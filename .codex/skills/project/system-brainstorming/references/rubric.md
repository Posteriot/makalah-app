# Brainstorming Rubric

Use qualitative scores only.

## Scale
- Potential: Low / Medium / High
- Effort: Easy / Medium / Hard
- Risk (per category): Low / Medium / High

## Risk categories
- Security
- Performance
- Compliance
- Cost
- Resource Dependency

## Scoring table (per idea)

| Idea | Potential | Effort | Security Risk | Performance Risk | Compliance Risk | Cost Risk | Dependency Risk | Notes |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |  |  |

## Notes
- Always list key assumptions and unknowns.
- If the data is insufficient, mark as "Unknown" in Notes and ask a targeted question.
