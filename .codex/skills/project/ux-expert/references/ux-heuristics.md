# UX Heuristics

## Clarity
- Tujuan halaman jelas dalam 5 detik.
- Label dan microcopy mudah dipahami.

## Feedback
- Semua aksi punya feedback (loading, success, error).
- Status proses terlihat jelas.

## Consistency
- Pola interaksi seragam di seluruh app.
- Komponen dan tone teks konsisten.

## Error Recovery
- Pesan error membantu user memperbaiki.
- Hindari dead-end.

## Accessibility
- Kontras, ukuran font, dan navigasi keyboard memadai.
- ARIA digunakan bila perlu.
