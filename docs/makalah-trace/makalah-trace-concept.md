# Makalah Trace: Kerangka Integritas Akademik

Status: Concept narrative (non-normative)

Makalah Trace adalah sistem audit proses yang menyajikan indikator keterlibatan intelektual siswa/mahasiswa saat menyusun paper bersama AI.
Posisinya adalah alat pertimbangan dosen/guru, bukan alat vonis otomatis.

> Landing message:
> "Gunakan Makalah Trace untuk melihat jejak keterlibatan intelektual secara transparan: kolaborasi aktif dengan AI, bukan sekadar output instan."

## Argumentasi Utama
1. Kampus/sekolah perlu jalan tengah antara larangan total AI dan ketergantungan penuh pada AI detector.
2. Dosen/guru butuh indikator proses, bukan hanya hasil akhir.
3. Mahasiswa/siswa butuh bukti keterlibatan intelektual yang dapat diaudit.

## Kerangka 4 Pilar (Konsep)
1. Originalitas Input (Inisiasi Ide)
2. Refleksivitas Kritis
3. Integritas Proses (termasuk sinyal dwell time)
4. Sinkronisasi Semantik (suara user terhadap output)

Pilar ini adalah kerangka evaluasi proses dan tidak berdiri sebagai vonis tunggal.

## Contoh Bentuk Output untuk Akademisi (Konseptual)
1. Label integritas dan ringkasan evidence per stage.
2. Score/confidence/label/summary/snippets.
3. Estimasi kontribusi user vs AI sebagai metrik tambahan (estimasi, bukan kepastian absolut).

Catatan:
1. Angka kontribusi dan metrik turunan adalah estimasi berbasis sinyal interaksi.
2. Semua hasil model tetap butuh interpretasi Editor.
3. Daftar output di bagian ini adalah contoh non-kontrak, bukan spesifikasi teknis final.

## Batasan Konseptual
1. Makalah Trace bukan alat penghukuman otomatis.
2. Indikator tidak menggantikan keputusan akademik dosen/guru.
3. Detail aturan akses, privasi, dan format output wajib mengacu ke dokumen acuan:
`docs/makalah-trace/editor-organization-management.md`.

## Posisi Produk
Makalah Trace memposisikan Makalah App sebagai partner riset yang dapat diaudit secara transparan dan bertanggung jawab.
