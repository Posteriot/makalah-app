# Naskah Feature

Folder ini menampung seluruh konteks kerja fitur `Naskah`.

Konvensi isi:
- `brainstorming/` untuk hasil eksplorasi ide dan keputusan awal
- `design/` untuk design doc
- `implementation-plan/` untuk rencana implementasi bertahap
- `verification/` untuk bukti verifikasi dan hasil cek

Penamaan fitur:
- Nama fitur/workspace: `Naskah`
- Nama tombol/entry point: `Buka Naskah`
- Nama mode internal kandidat: `main-document-workspace` atau `mainManuscript`
