# Success Metrics

## Prinsip
- Metrik harus terukur dan terkait outcome.
- Hindari vanity metrics.

## Contoh metrik per flow
- Aktivasi: user selesai onboarding.
- Engagement: user aktif di chat/paper workflow.
- Konversi: export berhasil atau upgrade (jika ada).

## Output
- Daftar metrik prioritas + alasan singkat.
