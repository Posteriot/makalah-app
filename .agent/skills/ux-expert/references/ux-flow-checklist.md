# UX Flow Checklist

## Alur Kritis
- Onboarding → auth → dashboard.
- Chat → paper workflow → export.
- Setting/admin bila ada.

## Friksi Umum
- Terlalu banyak langkah.
- Instruksi tidak jelas.
- State kosong tanpa guidance.

## Validasi
- Pastikan user tau posisi dan langkah selanjutnya.
- Pastikan ada jalan kembali (back/rewind).
